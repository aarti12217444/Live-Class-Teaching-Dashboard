# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

# 🎓 Live Class Teaching Dashboard

A modern full-stack Live Class Teaching Dashboard built using React.js, Node.js, Socket.IO, and Tailwind CSS. This platform is designed to help teachers conduct online classes efficiently while allowing students to join and interact in a structured learning environment.

---

## 🚀 Features

### 👨‍🏫 Teacher Module

* Teacher Dashboard
* Create and Manage Classes
* Live Class Interface
* Whiteboard Integration
* Student Management
* File Sharing Support
* Real-time Communication
* Attendance Monitoring (Extendable)

### 👨‍🎓 Student Module

* Join Live Classes
* View Shared Whiteboard

* Interactive Learning Interface

## 📥 Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/aarti12217444/Live-Class-Teaching-Dashboard.git
```

### 2️⃣ Navigate to Project

```bash
cd Live-Class-Teaching-Dashboard
```

### 3️⃣ Install Frontend Dependencies

```bash
npm install
```

### 4️⃣ Install Backend Dependencies

Move to server folder:

```bash
cd server
npm install
```

### 5️⃣ Start Backend Server

```bash
npm start
```

or

```bash
npm run dev
```

### 6️⃣ Start Frontend

Open a new terminal:

```bash
npm run dev
```

### 7️⃣ Open Application

```text
http://localhost:5173
```
